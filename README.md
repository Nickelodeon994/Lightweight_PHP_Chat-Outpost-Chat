# Lightweight_PHP_Chat | Outpost·Chat
超轻量级PHP聊天系统，无数据库，支持AI大模型

## ✨ 核心特性

- **零数据库依赖**：使用JSON文件存储数据，部署简单
- **AI机器人集成**：通过`@哨哨（关键词可自由修改）`召唤AI助手（默认为deepseek）
- **媒体支持**：支持图片（jpg/png/gif/webp）和视频（mp4/webm/ogg）
- **专为移动端设计**：目前未做桌面显示效果优化，暂时保留桌面端自动跳出
- **用户管理系统**：管理员审批机制，保障安全性
- **称号系统**：可自定义用户头衔，增强氛围感
- **深色模式**：支持手动切换或跟随系统主题

## 🚀 快速部署

### 环境要求
- PHP 8.0+
- 文件读写权限

### 安装步骤
1. 下载并上传至Web服务器
2. 确保 `data/` 目录可写
3. 访问 `index.php` 注册账户
4. 在data/users.json中将当前账户权限从user修改为admin，长按标题1.5秒即可进入管理后台
注意⚠️：data目录应当设置权限，确保只能本地读取，否则访客将能读取全部内容
